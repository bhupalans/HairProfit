# **App Name**: HairProfit

## Core Features:

- Purchase Input: Input purchase details including quantity and price.
- Processing Log: Record processing steps and associated costs.
- Cost Estimation: Calculate total costs including purchase price and processing expenses.
- Profit Calculation: Input selling price and calculate profit margin.
- Transaction History: Display the transaction history. It shows cost and selling price for the product.

## Style Guidelines:

- Primary color: Muted gold (#FFBF00) to evoke luxury and quality. This color complements hair products.
- Background color: Light beige (#F5F5DC) for a soft, neutral backdrop that allows the product details to stand out.
- Accent color: Deep brown (#A67b5b) to provide contrast and highlight important elements.
- Body font: 'Inter', a sans-serif, for a modern, objective feel. It can also serve as the headline font.
- Simple, elegant icons to represent different functionalities like purchase, processing, and sales.
- Clean and organized layout with clear sections for data input and display.
- Subtle transitions for a smooth user experience.
